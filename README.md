# RaidBuffet

Universal raid buff assignment system for World of Warcraft (TBC Classic).

## Features
- **Universal Grid:** Assign long-lasting buffs (>10 min) for Paladins, Priests, Mages, and Druids.
- **Click-to-Cast:** Cast assigned buffs directly from the UI.
- **Reagent Monitoring:** Automatically tracks required reagents (e.g., Arcane Powder) and warns you before you run out.
- **Global Synchronization:** All assignments are broadcasted and updated in real-time across the raid.
- **Native Localization:** Spell names are dynamically localized using the game's native API.

## Usage
Type `/rb` or `/raidbuffet` to toggle the main UI.
