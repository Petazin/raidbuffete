# RaidBuffet

Universal raid buff assignment system for World of Warcraft (TBC Classic).

## Features
- **Universal Grid:** Assign long-lasting buffs (>10 min) for Paladins, Priests, Mages, and Druids.
- **Smart Recommendations (Magic Wand):** Automatic combinatory assignment algorithm based on roster class sizes, specialization specs, and combat roles.
- **Click-to-Cast & Floating HUD:** Cast assigned buffs directly from the main UI or from the action-bar HUD overlay, which can be toggled/collapsed with a simple right click.
- **Main Tank Safety Checks:** Active detection of main tanks to prevent assigning Blessing of Salvation, warning you visually and whispering tanks with cooldowns.
- **Whisper Announcements:** Auto-whisper task assignments to group members with custom queueing and throttling anti-spam protection.
- **Reagent Monitoring:** Automatically tracks required reagents (including Druid rebirth seeds) and warns you locally, to group chat, or in capitals with sound alerts.
- **Global Synchronization:** All assignments are broadcasted and updated in real-time across the raid via P2P.
- **Native Localization:** UI and spell names are dynamically localized into Spanish and English.

## Usage
Type `/rb` or `/raidbuffet` to toggle the main UI.
